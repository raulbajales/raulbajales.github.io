Here is some of the work I did, or I was involved

Domain Specific Language

For SecurityScorecard

The requirement included implement a Domain Specific Language to write rules to analyse security related information about a domain or company and get a faceted resul as output, which in turns may be used to generate a security assessment report. This DSL must support hot reload, so an updated rule should not imply a new app deploy.
The implementation was done using Groovy.

Backoffice

For SecurityScorecard

The requirements included building a secure backoffice webapp for the customers, to be able to configure vendor domains to be scanned.
The implementation was done using MeanJS, using the Yeoman CRUD module generator, to rapidly generate views and controllers.

Real Time Sync Board

For Mural.ly

The requirements included the ability to sync in real time the state of a board (containing different types of assets, like images, post-its, etc.) between all the users currently in the same board.
The implementation included: Pusher to sync state between browsers, and NodeJS/ExpressJS/Mongoose to update shared board state (in background).

JSON Schema Validator in ActionScript 3

For Three Melons / Playdom

The requirement included a layer in client to validate that all the input coming from our servers were in a certain and specific JSON format.
The implementation included a component written in ActionScript 3, able to handle a preloaded registry containing a set of JSON files with JSON schemas. These Schemas were then used to validate incoming information from external servers.
The actual implementation was then published as Open Source: https://github.com/raulbajales/Frigga

Integrated Apache Shindig for the InApps platform

For LinkedIn

The requirements included building a component able to run apps inside LinkedIn, and these apps should be written using OpenSocial API.
The implementation included the integration of Apache Shindig (https://shindig.apache.org) as the foundation for the InApps Platform (the LinkedIn platform to host and run OpenSocial apps).
